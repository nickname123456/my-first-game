# План разработки игры на 5 дней

Цель плана - за 5 дней сделать завершенную версию **PM Simulator**, которую можно запустить, пройти от меню до результата и защитить по алгоритмам.

Главный приоритет: сначала рабочий игровой цикл и алгоритмы, потом визуальная полировка. Спрайты, звук и эффекты добавляются только после готовности основной логики.

## День 3 - A* и Behavior Tree сотрудников

**Цель дня:** сотрудники становятся автономными: ходят по офису и выбирают действия по ситуации.

### Что сделать

- [x] Представить офис как сетку для pathfinding.
- [x] Отметить на сетке:
  - [x] свободные клетки;
  - [x] стены;
  - [x] мебель;
  - [x] рабочие места;
  - [x] кухню;
  - [x] канбан-доску;
  - [x] переговорку.
- [x] Реализовать A*:
  - [x] открытый список через приоритетную очередь;
  - [x] восстановление пути;
  - [x] эвристика Манхэттенского расстояния.
- [x] Научить сотрудника двигаться по найденному пути.
- [x] Добавить целевые точки:
  - [x] рабочее место;
  - [x] кухня;
  - [x] канбан-доска;
  - [x] переговорка.
- [x] Реализовать минимальный Behavior Tree:
  - [x] `Selector`;
  - [x] `Sequence`;
  - [x] condition-узлы;
  - [x] action-узлы.
- [x] Реализовать поведение сотрудника:
  - [x] если выгорел - не работает;
  - [x] если есть кризис - идет в переговорку и ждет помощи;
  - [x] если устал - идет на кухню;
  - [x] если назанчили задачу - должен подойти в канбан, взять задачу и пойти на рабочее место;
  - [x] если есть задача - идет на рабочее место и работает;
  - [x] если задачи нет - гуляет по офису;
  - [x] Чем больше усталость сотрудника - тем медленнее он ходит и выполняет задачи.

### Алгоритмы дня

- [x] **A\*** - путь сотрудников по офису.
- [x] **Behavior Tree** - выбор действий сотрудников.

### Минимальный результат дня

- [x] Сотрудники не стоят статично всё время.
- [x] Сотрудник с задачей идет к рабочему месту.
- [x] Уставший сотрудник может пойти на кухню.
- [x] Сотрудник обходит препятствия, а не идет через стены.
- [x] Поведение выбирается через дерево, а не через один большой набор `if`.

### Тесты дня

- [x] A* находит путь вокруг препятствия.
- [x] A* возвращает пустой путь, если цель недоступна.
- [x] Behavior Tree выбирает работу, если сотруднику назначена задача.
- [x] Behavior Tree выбирает отдых, если усталость выше порога.
